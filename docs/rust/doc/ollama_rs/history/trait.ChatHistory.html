Expand description

A trait for managing chat message history.

This trait provides methods for adding messages to the history and retrieving the list of messages.

fn messages(&self) -> Cow<'_, [ChatMessage]>

Returns a reference to the list of chat messages in the history.

The messages are returned as a Cow (Clone on Write) to allow for efficient borrowing or cloning as needed.
